Use this plug-in to quickly create sample Deal & Coupon posts.

Set values for the define statements in tmt-sample-content.php.
	N: The number of posts you want to create.
	ACF_KEYs: the "Field Keys" from Custom Fields. To view the values, edit a ACF group, and select "Show Field Key" from screen options. Screenshot: http://screencast.com/t/jnIR9RWoje
